# 多模态内容风控系统（直播 / 图片）

这是一个用于 **直播画面** 和 **图片内容审核** 的小型风控系统 Demo。  
系统以“**先快后准**”为思路，先用轻量检测快速筛查，再在必要时做更深入判断。

适合用于：
- 直播内容合规检测
- 图片内容风控验证
- 风控 / 审核方向的工程展示

---

## 能做什么

- 识别 **二维码 / 联系方式 / 引流文字**
- 识别 **擦边 / 性暗示画面**
- 尽量减少普通直播话术的误报
- 输出清晰的审核结果和命中证据

---

## 整体流程（通俗版）

1. **第一步：快速检查**
   - 目标检测（YOLO）
   - 二维码快速识别
   - OCR 提取画面文字
   - 简单肤色比例判断（用于是否需要进一步检查）

2. **第二步：必要时再深入判断**
   - 当画面或文字“有点可疑”时，才会触发多模态模型进行兜底判断

3. **第三步：给出最终结论**
   - 通过 / 拦截
   - 风险评分
   - 命中的证据说明
